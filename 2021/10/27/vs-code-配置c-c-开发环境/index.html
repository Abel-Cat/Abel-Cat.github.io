vs code 配置c/c++开发环境

1 安装 Visual Studio Code

我们直接访问Visual Studio Code 官网，然后点击Download，进入页面后根据自己的操作系统选择对应的安装包。下载完成后，我们直接运行安装包，选择自己的安装路径，等待其安装完成即可。我们找到桌面上的图标后双击如果可以看到以下图的效果即为安装成功。

2 配置c/c++环境

2.1 安装vs code 的c/c++插件

我们在上图界面直接按下 Ctrl+Shift+X快捷键，在跳出的界面中输入c/c++，找到下图的插件，点击install。

3 特别提醒

如果我们不在源代码中的return前加上 system(“pause”); ，则会出现控制台界面一闪而过的情况，对于这种情况我们推荐使用两种方法来解决这个问题。

  1. 使用 system(“pause”);
  2. 在使用vsCode调试代码时在返回之前加上断点，然后就可以实现方法一类似的效果
